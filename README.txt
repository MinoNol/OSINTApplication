Ensure you have "anaconda" or "miniconda" as well as "git" installed.

- Create a conda environment, with Python 3.8 using "conda create --name osinit python=3.8".
- Clone repo with "git clone https://github.com/MinoNol/OSINTApplication"
- Intialise repo with "git submodule init" and "git submodule update".
- Run "pip install -r requirements.txt"
- Run "pip install -r maigret/requirements.txt"
- Run the program with the conda environment in VSCode.
